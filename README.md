# Law Practice Navigator

AI-powered strategic guidance for solo attorneys and small law firms.

## Version 0.1

This starter application includes:

- Next.js app structure
- Professional homepage
- Responsive layout
- Vercel-ready deployment configuration
- Project documentation folder

## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deployment

This project is ready to deploy on Vercel. Connect the GitHub repository to Vercel and deploy from the main branch.
